# 📅 Plataforma de Horarios & Métodos de Estudio con IA

Esta plataforma web permite a los estudiantes organizar sus horarios académicos de manera inteligente y generar métodos de estudio personalizados utilizando la **API de Gemini IA**. 

El proyecto utiliza una arquitectura de Monorepo dividida en `client` (Frontend) y `server` (Backend).

## 🚀 Tecnologías Utilizadas

### Frontend
- **React.js + Vite**
- **Tailwind CSS** (Estilos)
- **React Router** (Navegación)
- **Axios** (Peticiones HTTP)

### Backend
- **Node.js & Express**
- **MongoDB + Mongoose** (Base de datos)
- **JWT** (Autenticación)
- **Google Generative AI (Gemini)** (Lógica de IA)
